# PY-editX

This is my big project on PY. This project is about making some text editor on Python programming language.

## editX - A Feature-Rich Text Editor

editX is a simple yet powerful text editor built with Python and tkinter. It provides essential text editing features with an intuitive graphical user interface.

## Features

- **File Operations**: Create new files, open existing files, save and save as
- **Text Editing**: Cut, copy, paste operations
- **Undo/Redo**: Full undo and redo support
- **Find Functionality**: Search for text within your document
- **Keyboard Shortcuts**: Quick access to common operations
- **Auto-save Prompt**: Prompts to save changes before closing or creating new files

## Requirements

- Python 3.x (tkinter is included with standard Python installations)

## Installation

No installation required! Simply clone the repository and run the script.

```bash
git clone https://github.com/AzerUI1/PY-editX.git
cd PY-editX
```

## Usage

Run the text editor with:

```bash
python3 editx.py
```

Or make it executable and run directly:

```bash
chmod +x editx.py
./editx.py
```

## Keyboard Shortcuts

- **Ctrl+N**: New file
- **Ctrl+O**: Open file
- **Ctrl+S**: Save file
- **Ctrl+Shift+S**: Save as
- **Ctrl+Z**: Undo
- **Ctrl+Y**: Redo
- **Ctrl+X**: Cut
- **Ctrl+C**: Copy
- **Ctrl+V**: Paste
- **Ctrl+A**: Select all
- **Ctrl+F**: Find text
